CSS ( Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo (en-US) usada para descrever uma apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG , MathML ou XHTML ). Os requisitos de CSS como elementos são elementos na tela, no papel, na fala ou em outras mídias.

CSS é uma das principais linguagens da web aberta e é padronizada em navegadores web de acordo comespecificação da W3C. Desenvolvido em níveis, o CSS1 está atualmente obsoleto, o CSS2.1 é uma recomendação e o CSS3 , agora dividido em pequenos módulos, está progredindo para sua padronização.
